## Timelite Color System

The palette below captures the tones used on the Timelite homepage reference. Use these values to keep every new section consistent with the hero design.

| Token | Hex | Usage |
| --- | --- | --- |
| `backgroundMain` | `#F6EFE3` | Primary cream canvas, fills hero body background. |
| `backgroundLight` | `#FFFDF8` | Lighter cream for navbar/sections that need separation. |
| `textPrimary` | `#1F1A17` | Headlines and high-contrast text. |
| `textSecondary` | `#5F5B57` | Paragraph copy and muted navigation items. |
| `accentGold` | `#E2C675` | Small label text (e.g., "TIMELITE COUTURE"). |
| `accentGoldDeep` | `#B8953A` | Gold text on outlined CTA buttons. |
| `ctaPrimary` | `#B63A3F` | Filled call-to-action background (Shop Ao Dai). |
| `ctaPrimaryText` | `#FFFFFF` | Text on the filled CTA. |
| `ctaSecondaryBorder` | `#E4C768` | Outline color for the secondary CTA. |
| `ctaSecondaryText` | `#C29A3B` | Copy inside the outlined CTA. |
| `heroFrameBorder` | `#F1DA98` | Fine border around the oval hero image. |
| `iconStroke` | `#EBDFA1` | Strokes for circular icons in the navbar. |
| `whatsapp` | `#27BA55` | WhatsApp floating button. |
| `messenger` | `#1D64E2` | Messenger floating button. |
| `floatingAction` | `#F8DE5E` | Yellow close button in the floating contact tray. |

The values are approximated with the Timelite screenshot provided by the user. Adjustments can be made once design tokens are extracted from the original Figma/sketch files.
